# CausalTransPrediction

Simulation study for a biostatistics paper on **counterfactual risk prediction with transportability**, extended to a **meta-analysis** setting across multiple external databases.

---

## Files

| File | Description |
|------|-------------|
| `note.md` | Theoretical write-up: identification theorem, influence function derivation, double-robustness corollary, with full proofs. |
| `latex_code.tex` | Paper section covering the single-database simulation: DGP, plug-in and DR estimators, nuisance misspecification table. |
| `simulation.ipynb` | Single-database simulation. Runs all 16 correct/misspecified combinations of (m, π, ψ, ρ) across 500 Monte Carlo replications. Produces bias/RMSE tables and plots. |
| `simulation_meta.ipynb` | Three-database meta-analysis simulation. Each database has a unique DGP (confounders, treatment mechanism, outcome coefficients) but all identify the same β* = (1.0, 0.9, 0.7). Implements per-database DR, sandwich variance, and IVW fixed-effects pooling. |
| `table_results.tex` | LaTeX table of Monte Carlo results from the single-database simulation (bias and RMSE for all 16 scenarios). |
| `simulation_results.png` | Figure summarising single-database simulation results. |
| `convergence.png` | Convergence diagnostic plot. |
| `psi_rho_robustness.png` | Robustness plot for ψ and ρ nuisance misspecification. |
| `meta_efficiency.png` | *(generated by `simulation_meta.ipynb`)* Distribution of individual-DB vs pooled IVW estimates. |
| `meta_robustness.png` | *(generated by `simulation_meta.ipynb`)* Distribution of estimates under single-DB misspecification scenarios. |
| `CLAUDE.md` | Project brief for the AI assistant: theory, DGP parameters, open items. |
